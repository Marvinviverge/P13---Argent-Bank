***** POUR LANCER LE PROJET: *****

-- Vous devez dans un premier temps cloner le repo.--
-- Veillez à bien prendre le dossier backend actuel et suivre la procédure pour vous connecter à la BDD (voir .env)--
-- Concernant le frontend, une fois dans le dossier faîtes npm i .--
-- Une fois terminé, vous pouvez lancer le front avec la commande npm run starto .--
-- Le back se lance depuis son dossier avec la commande npm run dev:server .--
-- Vous pouvez maintenant vous connecter avec les identifiants fournis. --
